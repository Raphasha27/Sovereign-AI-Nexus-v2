# Sovereign AI Nexus v2

Experimental multi-agent AI orchestration platform with modular domain-specific agents.

## Tech Stack

- **Frontend:** React (Vite), JavaScript
- **Backend:** FastAPI (Python), Uvicorn
- **Modules:** AgriNexusAI, HealthNexusAI, CyberNexusSentinel, SolarGridSentinel, KasiLogistics, AquaNexusAI

## Features

- Domain-specific AI agents for agriculture, healthcare, cybersecurity, energy, logistics, and water management
- Centralized master command center
- Modular agent architecture for independent scaling
- Python-based backend orchestration

## Getting Started

```bash
git clone https://github.com/Raphasha27/Sovereign-AI-Nexus-v2.git
cd Sovereign-AI-Nexus-v2
# Backend
cd backend && pip install -r requirements.txt && uvicorn main:app --reload
# Frontend
cd frontend && npm install && npm run dev
```
